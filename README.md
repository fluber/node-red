# Node-RED: Opencart, mariadb, phpmyadmin, Opcua 

The purpose of this project is to enable students to quickly set up a testing environment for Node-RED and OpenCart.

## Overview

Using Docker to quickly set up virtual hosts.

## Installation

1. prepare install docker desktop application. https://www.docker.com/products/docker-desktop/
2. RUN docker-desktop application
3. change path to this folder under command mode.
4. docker-compose up -d, to run vritaul hosts.
